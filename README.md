# Creatorverse

## About

Creatorverse is a platform for discovering and sharing content creators.

## 🚀 Features

### 🛠️ Full CRUD Functionality
Manage your favorite content creators with a complete Create, Read, Update, and Delete system:
* **Add Creators:** easily onboard new talent with name, description, and social media links.
* **View Gallery:** A dynamic dashboard displaying all creators in a clean, responsive grid.
* **Edit Details:** Update creator information or social links on the fly with pre-filled forms.
* **Remove Creators:** One-click deletion to keep your 'verse' organized.

### ⚡ Technical Highlights
* **React & Supabase:** Seamless integration with Supabase for real-time data fetching and persistent storage.
* **Dynamic Routing:** Powered by `react-router-dom` for smooth transitions between the Dashboard, Creator Profiles, and Editor pages.
* **Smart Forms:** Centralized form components with `useEffect` state synchronization and unsaved change alerts.
* **Social Integration:** Direct external linking to YouTube, Instagram, and X (Twitter) profiles.
